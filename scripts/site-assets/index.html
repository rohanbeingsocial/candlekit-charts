candlekit/charts

The orchestration layer lightweight-charts leaves out — drawing tools, indicators, measurement, and deterministic replay — in one tree-shakeable, framework-agnostic package.

GitHub → npm MIT TypeScript React-optional
▸ Drawing tools — trendline · rectangle · fib · ruler
▸ Indicators — Bollinger · MACD · SMA · RSI
▸ Replay — deterministic bar-by-bar scrub

Live demos

▸ Vanilla JS
ChartController, series types (candles / line / area / volume), theme toggle. No framework.
▸ React
<ChartView> with series + timeframe + theme controls via the optional React bindings.
▸ Indicators
Built-in catalog — SMA, EMA, Bollinger, RSI, MACD, ATR, Stochastic — with a picker.
▸ Drawing tools
Trendlines, rays, rectangles, fib + the measurement ruler. Drag, select, persist.
▸ Replay
Deterministic bar-by-bar replay with play / pause / speed / scrub controls.

Quick start

# one peer dependency — no git URLs, no copyleft
npm install @candlekit/charts lightweight-charts
